# Sistema de Carrinho de Compras em Python

<div align="center">

  ![GitHub stars](https://img.shields.io/github/stars/username/repo)

</div>

Este projeto consiste em um sistema de carrinho de compras desenvolvido em Python, utilizando Programação Orientada a Objetos e listas e com uso da biblioteca random também.

O sistema simula uma compra em mercado, permitindo adicionar e remover produtos, calcular o total da compra, aplicar cupons de desconto e gerar uma nota fiscal ao final.

## ✨ Funcionalidades

- Geração automática de produtos com preço e estoque aleatórios
- Controle de estoque em tempo real ao adicionar produtos ao carrinho
- Remoção parcial ou total de produtos do carrinho
- Cálculo automático do valor total da compra
- Sistema de cupons aleatórios (0 a 3 disponíveis)
- Desconto aleatório entre 5% e 30% aplicado no total
- Geração de nota fiscal detalhada diretamente no terminal

## 📋 Prerequisites

Ter o Python instalado na máquina
Manter os arquivos main.py, produto.py e carrinho.py na mesma pasta


## 💻 Usage


Após executar o arquivo main.py, o sistema exibirá um menu interativo no terminal com as opções disponíveis.
O funcionamento ocorre da seguinte forma:

Digite 1 para adicionar produtos no carrinho.

Digite 2 para remover produtos do carrinho.

Digite 3 para ver o valor total de todos os produtos no carrinho.

Digite 4 para confirmar a sua compra

Digite 0 para encerrar o programa sem concluir nenhuma compra.
