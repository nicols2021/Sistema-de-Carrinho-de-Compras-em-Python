# Sistema-de-Carrinho-de-Compras-em-Python
Este projeto consiste em um sistema de carrinho de compras desenvolvido em Python, utilizando Programação Orientada a Objetos e listas e com uso da biblioteca random também.  O sistema simula uma compra em mercado, permitindo adicionar e remover produtos, calcular o total da compra, aplicar cupons de desconto e gerar uma nota fiscal ao final.
